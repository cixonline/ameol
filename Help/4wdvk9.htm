This CIX Forums script will retrieve a file that you would normally have to download using ftp and send it to you via an e-mail message. You will be asked to provide the name of the ftp site where the file can be found, the path of the directory in which it can be found and the name of the file. Your request will be added to a queue and you will receive the file via e-mail when your request has reached the top of the queue.
