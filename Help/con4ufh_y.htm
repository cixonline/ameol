Upload means to copy a file from your computer to an on-line service that you are connected to. If you have a file that you would like to send to another person on CIX Forums or add to the file list of a CIX forum, you will need to upload the file first.

You can upload directly to a forum by using the Upload File command in the CIX Forums Menu, or you can upload a file to your CIX Forums mail directory by using the Upload File command in the Mail menu.
