Check this box if you would like Ameol to save the original message before it decodes it. The message will be saved to the file 'atmailsv.dat' in your user directory (~\Ameol\users\<loginname>). This will allow you to re-import the original message or decode it with an external decoder. Please note that enabling this option will slow down the decode process and increase the amount of disk space used by Ameol.
