Command Line Switch

Syntax:      /auto

This command line switch forces Ameol to connect to CIX Forums using the blink settings contained in the Full blink action.
You can change the blink type used by specifying its' name, e.g. /auto="Forums Only" or /auto="Mail Only". Note that the name of the blink is case sensitive so /auto="FORUMS ONLY" or /auto="forums only" will not work.
