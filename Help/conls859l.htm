Rename Mail File Dialog Box
Previous  Top  Next


(Route : Mail Menu, Rename File Command)

The Rename Mail File dialog box allows you to rename a file that is held in your mail directory on CIX Forums.



If the file whose name you wish to change is not in the From text box, click the Filename button to select it from your Mail Directory file list.

In the To text box enter the new name for the file.

Click the OK button to rename the file the next time Ameol connects to CIX Forums.
