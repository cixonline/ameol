Resume
Previous  Top  Next


A resume is a text file that is held on CIX Forums that holds any details that you want to share with other users of the CIX system. For instance you may wish to include your address and telephone number, a career summary or just a few thoughts.

To create or edit your resume, select the Edit/Create My Resume command from the CIX Forums Menu. Once created and posted to CIX Forums any other user of the system can read this file. Similarly you can retrieve any other user's resume by selecting Show Resume from the message menu.
