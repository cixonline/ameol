(Route : CIX Forums Menu, File List Command)

This tab page shows a list of the files that are available for download from this CIX forum topic.



This is a Hot Spot image. Move your mouse pointer over the image
and when the mouse pointer changes to the shape of a hand click
the left mouse button once for extra help.

There are two or three tab pages on this dialog. The File List tab page shows the files that are available for download from the current forum topic. The Directory tab page shows all the files that are in the file directory for the forum topic. Not all of these files will be shown in the file list and you will only see this tab if you are the moderator of the selected forum. The Previous Downloads tab page shows the list of files that you have downloaded from this forum topic in the past.

Note that these lists are only as up-to-date as the last time they were retrieved from CIX Forums. To refresh the lists, click the Update button.

To download a file from the file list, go to the File List tab page and highlight the file you wish to retrieve, then click the Download button. You will see the File Download dialog box confirming the details of the file you wish to download. If these are correct click the OK button to retrieve the file the next time Ameol connects to CIX Forums.

If you want to search the file list, click the Find button.
