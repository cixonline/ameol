(You will see this dialog if you try to post a message as a Say in a Support forum).



The Support Forum Prompt dialog box warns you that if you try to post a message using the Say command in a Support forum on CIX Forums, your message may be withdrawn. The preferred method for creating a message in Support forums is by using the Submit Report command from the Help menu.

Click the Send Report button to send a report. You can post a Say by clicking the Use Say button.
