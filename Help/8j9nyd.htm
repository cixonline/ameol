Usenet is a global collection of discussion topics that are called newsgroups. The messages in these newsgroups are copied to news servers all over the world and anybody can subscribe to an open newsgroup.

Unlike CIX forums, the messages in newsgroups are only available for a number of days. This is because of the sheer quantity of messages that would build up in a very short space of time. If you want a complete set a messages from a newsgroup you will need to download new messages on a regular basis.
