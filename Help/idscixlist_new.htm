(Route : CIX Forums Menu, Show All Forums Command, New tab)

This dialog box displays any new CIX forums since the last list update.



This is a Hot Spot image. Move your mouse pointer over the image
and when the mouse pointer changes to the shape of a hand click
the left mouse button once for extra help.


Note that this list is only as up-to-date as the last time you retrieved the list. To get the latest version click the Update button, and the new list will be retrieved the next time Ameol blinks.

You can search the list by entering a word or string of characters in the text box at the top of the dialog. The list will then update to show only those forums that match your find term. To view the whole list again delete your search term from the text box.

The 'Last Message Date' column shows the date of the last message posted to that forum.

To join a forum, highlight the forum name you wish to join and click the Join button.

Some forums will have a 'lock' symbol next their name. These are closed forums, where the moderator of the forum has to join you.. To join a closed forum, you will need to send an e-mail to the moderator of the forum and ask to be added. You can find out who the moderators are by selecting the File Menu, the Script Item and choosing the Show Forum Moderators entry.

Any forum in which you already participate will be shown in red.
