(Route : Mail Menu, Directory Command)

The Mail Directory dialog box shows the files held in your mail directory on CIX Forums.



This is a Hot Spot image. Move your mouse pointer over the image
and when the mouse pointer changes to the shape of a hand click
the left mouse button once for extra help.


This dialog also allows you to apply commands to these files. All commands will take place the next time Ameol connects to CIX Forums.
This list is only as up-to-date as the last time it was retrieved from CIX Forums.

·   To refresh the list click the Update button.  

·   To download the currently highlighted file, click the Download button.  

·   To erase the currently highlighted file, click the Erase button.  

·   To send the currently highlighted file to another user on CIX Forums, click the Send File button.  

·   To export the currently highlighted file to a CIX forum topic, click the Export button.  

·   To rename the currently highlighted file, click the Rename button.
