Some CIX forum topics have file lists attached to them. Only the moderators of a forum can add or remove a file from these lists, so if you'd like a file to be included you will have to upload the file to the forum. Ameol will send a message to the moderators to tell them to include it in the list.

If Ameol holds a file list for a forum topic there will be an 'F' shown next to it in the In Basket.
