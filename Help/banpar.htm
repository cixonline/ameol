Ban Participant Dialog Box
Previous  Top  Next 


(Route : CIX Forums Menu, Participants, Ban Button)



The Ban Participant dialog box allows you to ban a participant from a conference that you moderate. Use the associated Unban Participant button to remove a ban from a banned participant.

Enter the CIX Forum nickname of the participant to be banned in the Enter participant name text box. If you highlighted a participant name in the Participants dialog box before clicking the Ban button then this box is pre-filled with the selected participant name.

Click OK to ban the participant the next time Ameol connects to CIX Forums.
