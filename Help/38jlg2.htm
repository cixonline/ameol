Script Command - String Prompt

Forum/Topic


Syntax:      %c('dialog caption', 'value')

Displays a forum list from which the user can select a forum and/or topic. The value field determines whether the list shows only forums or both forums and topics.



Value      Meaning

1      Display both forums and topics.

2      Display forums only



If the user presses the Esc key, the entire script is aborted and is not added to the Out Basket. If the user presses the Enter key, the highlighted topic or forum replaces the string prompt.

For example:



put 'mod %c('Add Participant','2')'



This displays a forums list showing only forums and with the caption 'Add Participant'. If the user selected the ameol_reg forum from the pick-list. then the entire string is updated to,



put 'mod ameol_reg'



This string is then output to the output script file which will subsequently be interpreted by Ameol when it comes to execute the entire script.
