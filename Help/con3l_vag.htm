This stage of the set up allows you to decide how you would like to connect to CIX Forums.

Choose Access CIX Forums via dial up if you'd like to use your modem to connect directly to CIX Forums.

Choose Access CIX Forums via the Internet if you'd like to connect to CIX Forums via your Internet connection.

If you choose to connect using either option then Ameol will initially connect via the Internet. You can change this later if you prefer.

When you have made your choice, click Next to continue.
