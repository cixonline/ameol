Your internet service provider may require separate login details for sending e-mail as for receiving it.

Check the "Sending e-mail requires authentication" box if your service requires a username and password to send e-mail.

The Login name text box requires your user name for the SMTP mail server.

The Password text box requires the password for the SMTP mail server. Check with your service provider if you're not sure about these details.
