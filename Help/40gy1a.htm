(Route : CIX Forums Menu, Participants Command, Participants Tab Page, Add Button)

The Add Participants dialog box allows you to add another user of CIX Forums to list of participants for this forum.



This is a Hot Spot image. Move your mouse pointer over the image
and when the mouse pointer changes to the shape of a hand click
the left mouse button once for extra help.


Please note that you can only add or remove participants if you are a moderator of the forum.

To add a participant, enter the CIX Forum nickname of the user you wish to add in the Name field. Now click the Add button to add their name to the list of participants to be added. You can remove a potential participant from this list by highlighting the name and clicking the Remove button. When you have finished adding participants to the list, click OK. The new participants will be added when Ameol next connects to CIX Forums.
