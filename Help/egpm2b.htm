This CIX Forums script allows you to send large text files (over about 30K). You can mail such a file to any e-mail address. Your file must be plain ASCII text, you cannot send other types of file using this script. You will be asked to locate the file you wish to send and the full e-mail address of the intended recipient. If the person that you are sending to is also a user of CIX Forums you can just enter their CIX Forum nickname.
